## 🔬 Overview

**BanglaLiteFormer** is a lightweight, purpose-built transformer architecture for binary sentiment classification of Bangla e-commerce product reviews. Designed for low-resource NLP settings, it achieves **97.00% test accuracy** with near-perfect prediction confidence (99.48%) — outperforming Bangla-BERT, GRU, LSTM, and transformer ensemble baselines — while using just ~150,000 parameters (vs. ~110M for Bangla-BERT).
